# mac快捷键

key | action
-- | --
option + 点击wifi图标 | 显示ip
长按最大化按钮 | 分屏
设置 -> 辅助功能 -> 鼠标与触控板 -> 触控板选项 | 启用三指拖移

# vscode快捷键
key | action
-- | --
option + shift + f | 格式化代码
command + k + f | 格式化代码
control + - | 回到上一次光标位置
option + shift + ↓ | 向下复制行

# vscode插件
插件 | 用途
-- | --
Prettier | 根据eslint格式化代码
